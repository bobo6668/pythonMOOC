## 6.1 集合类型及操作

集合 set

例如：
```python
A = { "python", 123, ("python", 123) }
```

注意：
空集合不能用 `{}` 生成，要用 `set()`。
因为 `{}` 生成的是空字典 `dict`。
```python
>>> de = {} ; type(de)
<class 'dict'>
>>> se = set() ; type(se)
<class 'set'>
```
